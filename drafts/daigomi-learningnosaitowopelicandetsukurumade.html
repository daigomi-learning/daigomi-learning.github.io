Daigomi LearningのサイトをPelicanでつくるまで

Pelican

Pelicanはインストール済みとする。

pelican-quickstart

必要なフォルダを作成する。

 mkdir -p static/{css, js, images}
 mkdir -p resouces

テーマのインストール

mkdir theme
cd theme
git clone https://github.com/DandyDev/pelican-bootstrap3.git
git branch daigomi
git checkout

pelicanconf.pyの設定

TIMEZONE = 'Asia/Tokyo'
DEFAULT_LANG = u'ja'

STATIC_PATHS = (['images'])

THEME = "theme/pelican-bootstrap3"
BOOTSTRAP_THEME = "cerulean"
CUSTOM_CSS = "static/css/style.css"

# Feed generation is usually not desired when developing
FEED_ALL_ATOM = None
CATEGORY_FEED_ATOM = None
TRANSLATION_FEED_ATOM = None

# Blogroll
LINKS =  ()

# Social widget
SOCIAL = (('twitter', 'http://twitter.com/lclj2013'),
          ('facebook', 'http://www.linkedin.com/pub/toshiyuki-takeda/26/9a0/881'),
          )

DEFAULT_PAGINATION = 10
DEFAULT_DATE = 'fs'
DATE_FORMATS = {
    'en': '%a, %d %b %Y',
    'jp': '%Y-%m-%d(%a)',
}
